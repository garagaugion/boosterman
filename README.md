# BOOSTERMAN Next.js

Versione Next.js pronta per Vercel del sito BOOSTERMAN.

## Comandi

```bash
npm install
npm run dev
```

## Deploy su Vercel

Imposta la variabile ambiente:

```text
OPENWEATHER_API_KEY
```

Il sito include:
- Google Ads tag `AW-862362843`
- Consent Mode
- API meteo `/api/weather`
- sitemap e robots automatici
- pagine SEO con metadata
